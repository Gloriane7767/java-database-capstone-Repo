## Admin User Stories
1. As an admin, I want to log into the system with a secure username and password, so that I can access administrative functionality safely.
2. As an admin, I want to add, update, or remove doctor accounts, so that I can keep the list of available doctors accurate and up to date.
3. As an admin, I want to add, update, or remove patient accounts, so that I can manage system access for patients.
4. As an admin, I want to view all scheduled appointments across the system, so that I can monitor overall clinic activity.
5. As an admin, I want to assign or revoke user roles (patient, doctor, admin), so that I can control who has access to specific functionality.
6. As an admin, I want to view system usage reports, so that I can track how the platform is being used and identify issues.
7. As an admin, I want to reset a user's password, so that I can help users who are locked out of their accounts.
8. As an admin, I want to deactivate a doctor's or patient's account, so that I can restrict access when necessary (e.g., policy violations).



## Patient User Stories
1. As a patient, I want to register for an account, so that I can access the system and book appointments.
2. As a patient, I want to log into my account securely, so that I can view and manage my personal health information.
3. As a patient, I want to search for available doctors by specialty, so that I can find the right doctor for my needs.
4. As a patient, I want to view a doctor's available time slots, so that I can choose a convenient appointment time.
5. As a patient, I want to book an appointment with a doctor, so that I can receive medical care.
6. As a patient, I want to view my upcoming and past appointments, so that I can keep track of my medical visits.
7. As a patient, I want to cancel or reschedule an appointment, so that I can adjust my plans when my availability changes.
8. As a patient, I want to view my prescriptions and medical records, so that I can stay informed about my treatment and health history.
9. As a patient, I want to receive a confirmation notification after booking an appointment, so that I know my request was successful.

## Doctor User Stories
1. As a doctor, I want to log into my account securely, so that I can manage my schedule and patient information.
2. As a doctor, I want to set and update my available time slots, so that patients can book appointments only when I am free.
3. As a doctor, I want to view a list of my upcoming appointments, so that I can prepare for patient visits.
4. As a doctor, I want to view a patient's medical history before an appointment, so that I can provide informed care.
5. As a doctor, I want to add or update a patient's prescription, so that the patient has an accurate and current treatment record.
6. As a doctor, I want to mark an appointment as completed, so that the system accurately reflects my appointment history.
7. As a doctor, I want to cancel an appointment if I am unavailable, so that patients can be notified and rebook accordingly.
8. As a doctor, I want to view my appointment statistics (e.g., total patients seen), so that I can track my workload over time.
